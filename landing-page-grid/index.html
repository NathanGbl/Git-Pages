Sobre mim

Olá, bem vindo a minha primeira página usando display grid. Sou Nathan Gabriel da Fonseca Leite e antes de continuar quero falar um pouco sobre minha trajetória até o momento na área de ciência da computação. Comecei a estudar nessa área em 2023 e, durante esse primeiro semestre, aprendi sobre os fundamentos de algoritmo usando python. Nessa matéria conheci os laços de repetição for e while, estruturas condicionais, operadores lógicos, listas, tuplas, dicionários, funções, arquivos, manipulação de string, módulos, pacotes e interface gráfica

Ainda sobre o primeiro semestre de 2023, outra matéria que tive na faculdade foi desenvolvimento web. Nela tive uma introdução ao HTML, CSS e JS, mas, além disso, tive canvas, servidor estático e dinâmico e banco de dados (usando Mongodb). Sobre o banco de dados, tivemos que instalar alguns pacotes pelo terminal como: express, ejs e mongodb na versão 4.12.
